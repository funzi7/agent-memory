## Confirmed Finnhub free-tier findings (exact field names)
- **G7 gross margin → `grossMarginTTM`** via `/stock/metric?metric=all`. Present &
  correct: AAPL 47.9%, MSFT 68.3%, NVDA 74.2% (also `grossMarginAnnual`,
  `grossMargin5Y`). Finnhub returns it as a **PERCENT** (`74.2`), so
  `finnhub_source.gross_margin()` divides by 100 → **fraction** (`0.742`);
  `metrics.gross_margin_pct` ×100 → `Gross_Margin%` (`74.2`); the gate compares vs
  `gates.gross_margin_min × 100` (default **40%**) → 74.2 ≥ 40 **passes**.
- **G8 quarterly revenue → `/stock/financials-reported?freq=quarterly`**. Returns
  ~8 quarters most-recent-first; revenue is read from each quarter's **`report.ic`**
  via a concept fallback chain: `us-gaap_RevenueFromContractWithCustomerExcludingAssessedTax`
  → `us-gaap_Revenues` → `us-gaap_SalesRevenueNet` → … → label. Namespace
  (`us-gaap_` / `us-gaap:`) is stripped. If no revenue concept matches → `None` →
  **"נתון רבעוני לא זמין"** (non-blocking, never guessed). The standardized
  `/stock/financials` endpoint stays premium and is NOT used.
