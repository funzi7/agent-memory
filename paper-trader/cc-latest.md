## How it works (read once/day — engine fills next-open)
- **Reader** (free, no paid X API): reads `@theaiportfolios`' ORIGINAL posts
  (retweets excluded) via **Nitter RSS**, trying a configurable instance list in
  order until one works. All instances fail → log + leave the portfolio UNCHANGED
  (no crash). Dedupe by post id in `state/twitter_state.json` (act only on posts
  newer than `last_id`; cursor advances past every post seen → re-runs never re-act).
- **Classify** (Claude API, `ANTHROPIC_API_KEY`, model in config): each new post →
  strict `{action:BUY|SELL|NONE, ticker, …}`. ONLY an explicit executed trade of a
  specific ticker counts; opinions / watchlists / ambiguity → NONE (never guess).
- **Strategy**: BUY adds the ticker then equal-weight rebalances (whole-share +
  small_account_fill, $100 participates); SELL fully exits the name only if held
  (else ignored); NONE ignored. Hebrew reasons `קנייה/מכירה לפי @<account>`. New
  `twitter_track_100/_10k` → **22 portfolios** (10 strategies ×2 + combined).
