## What this change did (dashboard polish — 5 fixes)
Committed directly to **agent-memory main** (public Pages host; no PR). SW cache
bumped **pt-dash-v8 → pt-dash-v9**. All bidi follows `paper-trader/RTL-RULEBOOK.md`.

1. **Summary-strip name overlap fixed (CSS).** "הטובה ביותר היום" / "החלשה ביותר היום"
   strategy names (e.g. `leveraged_momentum`) no longer collide. `.stat{min-width:0}`
   lets each grid cell shrink instead of overflowing into its neighbour; the name
   value gets `.v.name` (smaller, `word-break`) so a long identifier wraps onto its
   own line. Strip column min widened 150→165px.
2. **Trade-row reason cleaned.** Root cause: the naive CSV split broke the engine's
   quoted reason (`"rank 3, 6-1 +147.6%"` has an inner comma), leaking a stray `"`.
   Replaced `parseCSV` with a proper RFC-4180 parser (quotes + `""` escapes), so the
   reason stays intact. `reasonAscii` now relabels English `rank N` → Hebrew `דירוג N`
   (and `shortlist #N` → `דירוג סקרינר #N`, `6-1` → `מומנטום`) and strips stray
   `"`/`'`/gershayim. Applies to both the main trades table and the detail-page trades
   (and pending lines). Rendered row e.g. `2026-06-24 · ON · 🟢 קנייה · 17.0 · $117.19 · דירוג 5 · מומנטום +140.6%`.
3. **Header compacted.** `↺ אפס יעדים` + `⟳ רענון` now sit together on ONE compact
   row inside a `.actions` flex group (both `ghost`-styled, smaller padding/font),
   wrapping gracefully as a unit on phones.
4. **Entry + current price per holding.** Each holding row now shows `כניסה $X`
   (avg_price) and `נוכחי $Y` (latest close) alongside weight% and P&L%, on the cards
   AND the detail page. The detail table dropped its now-redundant "מחיר ממוצע" column
   (entry is in the row sub-line; "שווי" market value stays).
5. **Per-ticker rank.** Each holding shows its OWN `דירוג N`. Positions don't store a
   rank, so it's derived from the holding's latest BUY trade reason
   (`holdingRank` → `orderRank`: momentum "rank N", screener "shortlist #N"). Shown
   for momentum_scan / leveraged_momentum / screener_track; absent (correctly) for
   benchmark/rsi2/etc. Example row: `AMDL · דירוג 3 · כניסה $67.03 · נוכחי $66.74 · 19.7% · -0.49%`.

Files: `app.js` (parseCSV, reasonAscii, holdingRank, holdingsHTML), `dashboard.html`,
`strategy.html`, `style.css`, `sw.js`. Verified offline with a jsdom harness against
the live `portfolios.json`/`closes.json`/`trades.csv`; the RTL-rulebook regression
still passes (tickers isolated, $/signed LTR, Hebrew lines RTL).
